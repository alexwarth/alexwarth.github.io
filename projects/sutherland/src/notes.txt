-----

Sketchpad
* drawings: Drawing[]
* currentDrawing: Drawing
* scope: { offset: ..., zoom: ... }
* line/arc being drawn...
+ all UI code

-----

Drawing
* things: Thing[]
* attachers: Set<Handle> (from things)
* constraints: ConstraintSet
+ deleteThingAt(pos)
  - finds a thing, deletes it
  - removes any constraints that are about variables or handles that no longer exist
+ multiplyScaleOfThingAt(pos, factor)
+ merge(oldHandle, newHandle)
  - updates all things (refs to oldHandle are now to newHandle)
  - updates all constraints, possibly discarding some (in case they become duplicates)
+ cleanUp()
  - removes constraints that are about variables or handles that are no longer around
  - removes attachers that were associated w/ handles from master that are no longer there

+ forEachVar(...)
